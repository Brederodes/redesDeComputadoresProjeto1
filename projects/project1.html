EXERCÍCIO AULA 1: CLIENTE-SERVIDOR

 Um protocolo pode ser entendido como o conjunto de regras para comunicação entre máquinas na rede. Neste projeto, foi criado o seguinte protocolo para uma simples rede cliente-servidor:

 O cliente pode enviar uma array com três itens para um servidor, sendo os dois primeiros números que serão utilizados em ordem como os parâmetros de uma operação determinada pelo terceiro item. O servidor, ao receber a array, devolve um número que é o resultado da operação realizada.

 O objetivo da existência dessa rede é a terceirização de poder computacional, fazendo o computador "cliente" ser um modo de acessar o poder computacional do servidor.

MENSAGEM
1
2
0
RESPOSTA
3
O terceiro número significa, respectivamente:
  • 0 - adição
  • 1 - multiplicação
  • 2 - divisão
